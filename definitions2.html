Definitions Group Two

The three major programming control structures are:
Sequence: This is the most basic control structure. It tells the computer to execute the statements in order.
Selection: This control structure allows the computer to choose between different statements based on a condition.
Iteration: This control structure allows the computer to repeat a set of statements until a condition is met.

Hypertext markup language (HTML) is the language used to create web pages. It is a markup language, which means that it uses tags to tell the browser how to display the content of a web page
Markup languages are used to describe the structure and content of a document. They are used to create documents that can be displayed on a variety of devices, including web browsers, email clients, and word processors.

Cascading style sheets (CSS) is a language used to style the content of a web page. It is used to control the appearance of elements on a web page, such as the font, color, and size of text.

Syntax and uses of programming languages are the rules that govern how a programming language is written and interpreted. The syntax of a programming language defines how the statements in the language are structured and how they are interpreted by the compiler or interpreter. The uses of programming languages vary widely, but they are typically used to create software applications, websites, and other computer programs.
