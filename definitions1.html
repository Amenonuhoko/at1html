Definitions Group One

Authentication is the process of verifying a user's identity. Web security is the practice of securing web applications and websites from unauthorized access, use, disclosure, disruption, modification, or destruction.

HTTP is the set of rules for how data is transferred between a web server and a web browser. It is a stateless protocol, which means that each request and response is independent of the previous request and response.

Session management is the process of tracking the state of a user's interaction with a web application. This includes things like the user's login status, the pages they have visited, and the items they have added to their cart.

Stateless programming is a programming paradigm in which each request from a client to a server is treated as a separate event. This means that the server does not store any information about the client's previous requests.
